ข้อมูลหลักสูตร

  • หน้าหลัก

  • ตารางรายวิชา

ชื่อหลักสูตร : วิศวกรรมศาสตรบัณฑิต สาขาวิชาวิศวกรรมสารสนเทศ และเครือข่าย

รหัสหลักสูตร : 67062043

ระดับการศึกษา : ปริญญาตรี

คณะ / มหาวิทยาลัย : คณะเทคโนโลยีและการจัดการอุตสาหกรรม / มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ วิทยาเขตปราจีนบุรี

คำอธิบาย : เป็นสาขาหนึ่งของวิศวกรรมศาสตร์ที่เน้นการศึกษาและประยุกต์ใช้ความรู้ด้านคอมพิวเตอร์ เครือข่าย และเทคโนโลยีสารสนเทศ เพื่อออกแบบ สร้าง และจัดการระบบเครือข่ายคอมพิวเตอร์ รวมถึงการพัฒนาซอฟต์แวร์และโซลูชันที่เกี่ยวข้องกับระบบเครือข่าย
