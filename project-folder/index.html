  • ประวัติส่วนตัว
  • หน้าหลัก
  • ตารางรายวิชา

ชื่อหลักสูตร: วิศวกรรมศาสตรบัณฑิต สาขาวิชาวิศวกรรมสารสนเทศ และเครือข่าย

รหัสหลักสูตร: 25620151100293

ระดับการศึกษา: ปริญญาตรี 3 ปี

คณะ / มหาวิทยาลัย: คณะเทคโนโลยีและการจัดการอุตสาหกรรม / มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ วิทยาเขตปราจีนบุรี

คำอธิบาย: หลักสูตรที่เน้นการพัฒนาความรู้ด้านโปรแกรมมิ่ง อัลกอริทึม และเทคโนโลยีคอมพิวเตอร์ที่ทันสมัย
